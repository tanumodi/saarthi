About Saarthi

How it works

Saarthi is a solution based on GIS Based Images of the potholes/Landslides/Accidents/Other Hazards which are captured by the citizens. It connects the citizen by providing a space to report such problems by posting images of exact location. Government has the access to all the information provided and hence can resolve it as soon as possible . In return citizens do get reward points(which provides some privileges on toll-taxes etc) when a particular reported problem has been verified to be correct by the government.

  • Works for ios & Android
  • GIS based images
  • Rewards for accepted reports
  • Problem detection easy for government
